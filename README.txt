Para pasar los pdf a una página html con la nube de las palabras que más se repiten se hace lo siguiente:
1. Se introduce en la carpeta "fichas_docentes" todas las fichas docentes que se quieran pasar. Deben tener formato pdf, ya que es el único formato admitido.
2. Se ejecuta el programa "pdf2htmlWordCloudAll.js".
3. En la carpeta "nubes_ficha_docente" se encuentran todas las páginas html con las nubes de las palabras que más se repiten en cada pdf.
4. La página html "complutense.html" es un ejemplo de como integrar las nubes a la página principal de fichas docentes de la Universidad Complutense de Madrid
